## File Naming Convention

**Standard format:**
- `[BRAND]_logo.png` — Full logo with wordmark
- `[BRAND]_icon.png` — Icon/symbol only
- `[BRAND]_logo_dark.png` — For light backgrounds
- `[BRAND]_logo_light.png` — For dark backgrounds
- `[BRAND]_wordmark.png` — Text only, no icon

- TB10X (or TBTX) = TransformBy10X
