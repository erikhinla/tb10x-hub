# TB10X Canon Language Guardrails

Use this note when writing or revising campaign and architecture language in this repository.

## Canonical Definition: FLOW Agent OS

**FLOW Agent OS means Agent Orchestration System.**

In TB10X language, OS = **Orchestration System**, not a device or computer operating system.

FLOW Agent OS coordinates context, task routing, agent sequencing, workflow orchestration, execution state, and outputs across tools and agents. It is the conductor of execution, not the machine itself.

## One-Line Version

FLOW Agent OS is an **Agent Orchestration System** that coordinates execution across tools, workflows, and agents.

## Architecture Placement (System Language)

- **TransformBy10X**: worldview, thesis, and architecture framing.
- **BizBuilders AI**: infrastructure build and implementation layer.
- **BizBot Marketing**: execution and growth application layer.
- **BizBrain Lite (control plane)**: coordination, memory, registry, and runtime state.
- **JET STREAM**: job envelope tracking stream.
- **FLOW Agent OS**: execution coordination layer that orchestrates routed work on top of shared system state.

## Banned or Incorrect Phrasing

- FLOW Agent OS as desktop/device Operating System
- replacement for Windows / macOS / Linux
- device-level OS framing
- generic “AI operating system” phrasing without orchestration context
- generic software platform language that hides orchestration role
- Deploy FLOW / FLOW Agent OS product-page framing
- assessment / diagnostic / audit / readiness score language unless explicitly approved
- SaaS pricing-plan framing when not explicitly approved

## Usage Notes

- Always expand intent in first mention: **FLOW Agent OS (Agent Orchestration System)**.
- After first mention, “FLOW Agent OS” or “orchestration layer” is acceptable.
- Keep FLOW Agent OS inside architecture context, never as a stand-alone device-OS claim.
- State what it coordinates: context, tasks, agents, workflows, outputs.
- If uncertain, pause and request copy approval before introducing new phrasing.
