## Export Workflow

### Phase 1: Manual Export (Current)

1. Open `/social` hub in browser
2. Select template type
3. Select headline
4. Select support line
5. Select series label
6. Switch to export stage
7. Choose aspect ratio preset
8. Click "Export PNG"
9. Save to appropriate campaign folder
10. Optional: Polish in Canva if needed

### Phase 2: Future Enhancements

- Auto-suggest folder based on series lane
- Batch export (multiple ratios at once)
- SVG export for text-led assets
- Direct upload to media library

## Asset Naming Convention

**Format:** `[template-type]-[ratio]-[variant].png`

**Examples:**
- `quote-card-1080x1350-v1.png`
- `daily-truth-1080x1920-tool-drool.png`
- `architecture-files-1920x1080-infra-explainer.png`
